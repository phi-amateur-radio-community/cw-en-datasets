# CW-EN Datasets

AI datasets of CW-EN.

## Format

### Data Format

`{"cw": <cw content>, "en": <en content>}`

### File Format

The length of a single file is limited to 100 lines
The file is named `pced_xxxx.jsonl`, such as `pced_0021.jsonl`
